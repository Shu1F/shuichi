# UI Guidelines

## Design Principles

- Minimal and typography-first design
- Use whitespace as a primary element
- Avoid unnecessary visual noise

## Constraints

- Do not introduce new colors beyond existing tokens
- Do not add shadows, gradients, or decorative UI
- Do not create card-style components
- Keep layouts simple and linear

## Implementation Rules

- Use existing layout and UI components (PageFrame, Header, Footer)
- Do not redefine spacing or sizes unless necessary
- Follow existing patterns in the codebase
